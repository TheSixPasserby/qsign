# 如何更换仓库？
``` bash
    cd ./.git
    vim config
    url = https://github.com/TheSixPasserby/qsign.git
```
或
```bash
    git remote set-url origin https://github.com/TheSixPasserby/qsign.git
```
然后进入脚本更新即可
